This is a Simple Blog Website built using Express.js and EJS that demonstrates full CRUD (Create, Read, Update, Delete) operations.
The application supports two types of users:

#Reader User

Can view and read blog posts only

#Publisher User

Can create new blog posts, 
Can update existing posts, 
Can delete blog posts
#Session based authentication

All blog data is stored and managed using MongoDB

#Purpose

Learn backend fundamentals

Understand how an Express server connects to MongoDB

Practice CRUD operations using MongoDB
